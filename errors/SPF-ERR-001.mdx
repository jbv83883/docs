## Debug Issues

1. **Download the Debug Tool**
    - Download our debug tool to identify issues on your device

2. **Run the Debug Tool**
    - Extract all files from the ZIP archive.
    - Run the tool as an administrator.
    - Any text in red indicates an issue that needs to be fixed.

<Warning>
Ensure you run the tool with administrator privileges to get accurate results.
</Warning>

## Run Repair Scripts

1. **Download the Repair Scripts**
    - Download the Auto Fixes ZIP archive.

2. **Extract and Run Scripts**
    - Extract all files from the RAR archive.
    - Run each script as an administrator.
    - If prompted, type "Yes" to proceed.
    - After running all scripts, restart your device.

<Note>
Running these scripts as an administrator is crucial for them to function properly.
</Note>
